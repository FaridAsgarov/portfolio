Mənim adım Farid Asgarov

Salam! Mənim adım Fərid Əsgərov və mən Avstriyanın Vyana şəhərində yaşayıram. Vyanada yaşamadan əvvəl, bir neçə illər İngiltərənin London şəhərində yaşadım, orada Qlobal Finans İdarəetmə Bakalavrı bitirmişəm. Mən Azərbaycan, İngilis, Rus, və Alman dillərində danışa bilirəm.

Mən hazırda Veb geliştirici bacarıqlarım üzərində işləyirəm. HTML 5, CSS 3, JS, JQuery və React kimi Front-End İnkişaf dilləri də daxil olmaqla bir çox fərqli dildə proqramlaşdırma təcrübəm var. MySQL və PHP ilə Back-end tərəfdən proqramlaşdırma ilə bağlı bir az təcrübəm var.

Veb inkişaf proqramlaşdırman basqa, Python və Java ilə proqram təminatı ilə də maraqlanıram. Hal-hazırda Java proqramlaşdırmasını öyrənirəm və belə vacib proqramlaşdırma dilini öyrənmək imkanından çox maraqlanıram.
